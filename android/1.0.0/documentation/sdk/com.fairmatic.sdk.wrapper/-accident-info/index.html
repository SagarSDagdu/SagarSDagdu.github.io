Content copied to clipboard

Measures an approximate precision of the detected collision. Ranges between 0 to 100. A confidence number of 0 indicates that it was not a collision. This is never sent as part of com.fairmatic.sdk.wrapper.FairmaticBroadcastReceiver.onPotentialAccident. It is sent as part of com.fairmatic.sdk.wrapper.FairmaticBroadcastReceiver.onAccident to invalidate a previous onPotentialAccident callback. Checkout com.fairmatic.sdk.wrapper.FairmaticMockAccidentConfiguration.shouldInvalidateFinalCallback to test this scenario during development.
