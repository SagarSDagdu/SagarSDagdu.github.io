FairmaticVehicleInfo

fun FairmaticVehicleInfo(vehicleId: String, bluetoothAddress: String)
Content copied to clipboard

Parameters

vehicleId

The id of the vehicle to be associated with the SDK. Each vehicle associated with the SDK needs a unique id. This id will be returned in AnalyzedDriveInfo if the device was connected to the vehicle's bluetooth during a drive. Please use DriveInfo.vehicleTaggingDetails to get the details of the vehicle tagged. The id may be upto 64 characters long. You can use com.fairmatic.sdk.Fairmatic.isValidInputParameter to validate the vehicle Id. Passing an invalid id to FairmaticVehicleTagging.associateVehicle will result in an error.

bluetoothAddress

The hardware address of the vehicle's bluetooth to be associated with the SDK. This address will be used to scan for the vehicle during drives. The Bluetooth device can be found either through device discovery or by querying the list of paired devices. Please refer to this guide for more details. The address can be fetched by calling FairmaticVehicleInfo.getBluetoothAddress on the corresponding BluetoothDevice object. Passing an invalid bluetooth address to FairmaticVehicleTagging.associateVehicle will result in an error. You can use BluetoothAdapter.checkBluetoothAddress(java.lang.String) to validate the bluetooth address.
