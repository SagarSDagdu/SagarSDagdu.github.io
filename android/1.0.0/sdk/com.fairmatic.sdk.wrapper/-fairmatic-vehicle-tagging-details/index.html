The vehicle tagging details of a drive. These details are provided for drives that are tagged by a vehicle. For a given vehicle, both bluetooth car stereo and beacon can be used to tag with the same vehicleId. In case a drive is tagged using both bluetooth car stereo and beacon having different associated vehicleIds, the bluetooth car stereo vehicleId supersedes the beacon vehicleId.
